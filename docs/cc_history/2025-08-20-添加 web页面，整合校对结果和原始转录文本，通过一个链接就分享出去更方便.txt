     ## 功能概述
     本系统现已支持为每个转录任务生成永久的web查看链接，用户可以通过链接在浏览器中查看转录结果，无需通过企业微信接收长文

     - **永久有效**：查看链接永不过期

  📝 使用方法

  1. 修改配置文件
  {
    "web": {
      "base_url": "https://your-domain.com",  // 修改为你的实际域名
      "enable_view_links": true
    }
  }
  2. 启动服务
  python main.py --start
  3. 提交任务后自动生成查看链接，格式为：
  https://your-domain.com/view/view_xxxxxxxxx

  🔧 配置示例

  生产环境：
  "web": {
    "base_url": "https://transcript.example.com",
    "enable_view_links": true
  }

  开发环境：
  "web": {
    "base_url": "http://localhost:8000",
    "enable_view_links": true
  }
